CSS-свойства: Border, Outline, Border-Radius, Box-Shadow, Opacity, Visibility и Background

CSS-свойства: Border, Outline, Border-Radius, Box-Shadow, Opacity, Visibility и Background CSS — это основа для оформления внешнего вида веб-страниц. Различные свойства позволяют задавать стили для элементов, такие как рамки, тени, прозрачность и фон. В этой статье разберем несколько ключевых CSS-свойств, таких как border, outline, border-radius, box-shadow, opacity, visibility и background, и рассмотрим, как они применяются на практике.

  • 2. Outline (Контур)
  • 3. Border-radius (Скругление углов)
  • 4. Box-shadow (Тень блока)
  • 5. Opacity (Прозрачность)
  • 6. Visibility (Видимость)
  • 7. Background (Фон)

1. Border (Граница)

Свойство border задает рамку вокруг элемента.

Синтаксис:

border: 2px solid #000;
  • Ширина: 2px — толщина границы;
  • Стиль: solid — тип границы, который может быть solid (сплошная), dashed (пунктирная), dotted (точечная) и др.
  • Цвет: #000 — цвет границы в формате HEX (чёрный).
Можно также задавать границы по сторонам отдельно: border-top, border-right, border-bottom, border-left.

Пример:

border-top: 4px dashed #f00;

2. Outline (Контур)

Свойство outline похоже на border, но оно не влияет на размер и положение элемента, так как размещается поверх рамки и не влияет на размеры.

Синтаксис:

outline: 3px dotted #00f;
  • Ширина: 3px — толщина контура;
  • Стиль: dotted — тип линии (в данном случае точечный);
  • Цвет: #00f — синий цвет.

3. Border-radius (Скругление углов)

Свойство border-radius позволяет скруглить углы элемента, создавая эффект закруглённой рамки.

Синтаксис:

border-radius: 10px;
  • Радиус: 10px — радиус скругления углов.
С помощью этого свойства можно создавать как симметричные, так и асимметричные углы, задавая отдельные значения для каждой стороны (например,border-radius: 10px 20px 30px 40px;).

Пример:

border-radius: 50%;
Этот код создаёт круг, если элемент имеет одинаковую ширину и высоту.

4. Box-shadow (Тень блока)

Свойство box-shadow задает тень для элемента.

Синтаксис:

box-shadow: 5px 5px 15px rgba(0, 0, 0, 0.3);
  • Горизонтальное смещение: 5px — смещение тени по горизонтали;
  • Вертикальное смещение: 5px — смещение тени по вертикали;
  • Размытие: 15px — радиус размытия тени;
  • Цвет тени: rgba(0, 0, 0, 0.3) — цвет тени с прозрачностью 30%.
Пример:box-shadow: 0 0 10px #00f;

5. Opacity (Прозрачность)

Свойство opacity задает степень прозрачности элемента.

Синтаксис:

opacity: 0.8;
Значение opacity задается числом от 0 до 1, где 0 — полная прозрачность (невидимый элемент), а 1 — полное отсутствие прозрачности (непрозрачный элемент).

Пример:

opacity: 0.5;
Этот код сделает элемент полупрозрачным.

6. Visibility (Видимость)

Свойство visibility управляет видимостью элемента.

Синтаксис:

visibility: hidden;
Значение hidden делает элемент невидимым, но сохраняет его место в документе. В отличие от свойства display: none, которое полностью убирает элемент с страницы, visibility: hidden просто скрывает его. Пример:
visibility: visible;
Элемент будет видимым.

7. Background (Фон)

Свойство background задает фон элемента. Оно поддерживает множество значений: цвет, изображения, градиенты и другие.
background: #f0f0f0;

Цвет фона: #f0f0f0 — светло-серый цвет.

Можно использовать изображение как фон:

background: url('image.jpg') no-repeat center/cover;
  • Изображение: url('image.jpg') — ссылка на картинку;
  • Повтор: no-repeat — изображение не повторяется;
  • Позиция: center — изображение центрируется;
  • Размер: cover — изображение масштабируется по размеру блока.
background: linear-gradient(to right, red, yellow);
Этот код создаёт градиент от красного к жёлтому слева направо.

Заключение:

Эти CSS-свойства позволяют контролировать внешний вид элементов на веб-странице.

Свойства border и outline создают рамки вокруг элементов, border-radius добавляет скруглённые углы, box-shadow позволяет создавать тени, а opacity и visibility управляют прозрачностью и видимостью. Свойство background задаёт фон элемента, будь то цвет, изображение или градиент. Используя эти свойства вместе, можно создавать элегантные и визуально привлекательные веб-интерфейсы.
